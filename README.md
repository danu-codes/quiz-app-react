# 🧠 React Quiz App

A simple and interactive quiz application built using **React**.

This project was created to practice React fundamentals including state management, hooks, conditional rendering, and dynamic UI updates.

Users can answer questions within a time limit, track their performance, and review answers after completing the quiz.

---

## 🚀 Features

✅ Start screen before quiz begins  
⏱ Countdown timer for each question  
📊 Score tracking system  
❓ Dynamic questions loaded from JSON  
🎯 Instant answer selection  
🔄 Automatic question change when time expires  
📝 Answer review after completing the quiz  
✅ Correct and wrong answer tracking  
📈 Accuracy percentage calculation  
⌛ Total quiz completion time tracking  
🔁 Restart quiz functionality  
📱 Responsive UI design

## 🎮 How It Works

1. Click **Start Now** to begin the quiz
2. Answer each question before the timer ends
3. Your score updates automatically
4. After finishing:
   - View score
   - Check correct/wrong answers
   - See accuracy percentage
   - View total time taken
